### coff

coff is my bootstrapped compiler toolchain. i wrote the first version (non-bootstrapped general purpose language) in 2020, using python, and since then i have rewritten it... i think five times? i have honestly lost count. the current version is a complete rewrite that i started in 2026, using assembly. it now compiles itself and has grown into a systems programming language.

---

### moonshot

moonshot is my operating system, mostly written in **c0**, the language implemented by coff. it is currently made up of a bunch of smaller projects: the kernel (**knekt**), the memory management system (**jenna**), the scheduler (**chrone**), the filesystem (**jakel**), the networking stack (**verk**) and the framebuffer and font renderer (**punkt**). it is currently bootable on bare metal, which still feels a little surreal to say.
