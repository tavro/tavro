### coff

coff is my bootstrapped compiler toolchain. i started working on it back in 2019, and since then i have rewritten it... i think five times? i have honestly lost count. the current version is a complete rewrite that i started in 2026. it now compiles itself and has grown into a systems programming language.

---

### moonshot

moonshot is my operating system, mostly written in **c0**, the language implemented by coff.

it is currently made up of a bunch of smaller projects:

- **knekt**: the kernel
- **jenna**: memory management
- **chrone**: the scheduler
- **jakel**: the filesystem
- **verk**: the networking stack
- **punkt**: the framebuffer and font renderer

it is currently bootable on bare metal, which still feels a little surreal to say.
